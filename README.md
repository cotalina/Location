# Location
# 自动定位跟踪，定时邮件发送，适合给老人小孩使用<br> <br> 

已测试多部Android5.0及以上的机型(有问题反馈吧)<br> <br> 

思路：     1.先获取手机定位网络权限
 
          2.设置获取定位的间隔时间与接收邮箱

          2.开启service

          3.获取经纬度 逆向转换获取地址

          4.通过邮箱发送
<br> <br> 

ps：具体自己看源码。<br> <br>
